# Chart Intelligence Module — Tactical Calm v1

## Purpose
Renders an educational D3.js candlestick chart that detects key candle types
and chart patterns, then explains them in calm, plain English.

## Installation
1. Copy this folder to `/src/modules/chart-intelligence/`.
2. Add a route `/chart` that renders `ChartIntelligencePage`.
3. Ensure D3.js is available in the project.
4. Start the dev server — the chart will render using mock AAPL & TSLA data.
